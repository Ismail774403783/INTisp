One Command, Does it all

We wanted to make IntISP so easy to use, that you only had to run one command. Yes you heard that right one command, other control panels usually have more than one.


Warning! Before running this command please make sure you have a fresh installation of ubuntu as it is recommended.
$ sudo apt-get update && sudo apt-get install git make && git clone https://github.com/INTisp/INTisp.git && cd INTisp && sudo make
